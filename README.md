Brotatolike 2D射击生存游戏

一个基于Unity引擎开发的2D射击生存游戏，类似于Brotato（土豆兄弟）玩法，玩家需要在一波波敌人的攻击下生存下来，收集资源升级角色和武器。

项目介绍

这是一个独立开发的2D射击生存游戏，玩家控制角色在2D场景中生存，击败不断涌现的敌人，收集金钱和经验升级，解锁新角色和武器，挑战更高难度。

核心功能

 🔫 多样化武器系统：多种武器类型，每种武器有独特的攻击方式和升级路径
 🎮 多角色选择：不同角色有独特属性加成，可解锁新角色
 📊 属性成长系统：通过收集资源升级角色属性
 🤖 智能敌人AI：多种敌人类型，不同行为模式
 💰 经济系统：金钱收集和商店购买机制
 📈 经验升级系统：击败敌人获得经验，升级解锁新能力
 🎯 2D战斗系统：多种子弹类型和射击逻辑

核心功能实现流程

1. 游戏启动流程
```
1. 加载GameManager单例
2. 读取JSON配置文件（角色、武器、敌人、难度、道具）
3. 初始化角色选择界面
4. 玩家选择角色后进入游戏场景
5. 初始化敌人生成系统和商店系统
```

2. 战斗流程
```
1. 敌人生成系统按波次生成敌人
2. 玩家控制角色移动和射击
3. 子弹与敌人碰撞检测
4. 敌人受伤/死亡处理
5. 敌人死亡掉落金钱
6. 玩家收集金钱和经验
7. 达到一定波数后开启商店
```

3. 升级流程
```
1. 玩家收集经验值
2. 经验值达到阈值后升级
3. 显示升级选项（属性提升或武器解锁）
4. 玩家选择升级项
5. 更新角色属性或解锁新武器
```

主要类

GameManager
 功能：游戏核心管理器，管理游戏状态、资源和全局数据
 主要方法：
 `Awake()`：初始化游戏资源和数据
 `InitProp()`：初始化角色属性
 `FusionAttr()`：属性融合机制
 `GetRandom<T>()`：随机获取列表元素

PlayerController
 功能：玩家角色控制器，处理移动和射击逻辑
 主要方法：
 `Update()`：更新玩家状态和输入处理
 `Shoot()`：处理射击逻辑
 `Move()`：处理角色移动

EnemyController
 功能：敌人AI控制器，处理敌人行为
 主要方法：
 `Update()`：更新敌人状态和AI决策
 `ChasePlayer()`：追击玩家逻辑
 `AttackPlayer()`：攻击玩家逻辑

WeaponSystem
 功能：武器系统，管理武器切换和升级
 主要方法：
 `SwitchWeapon()`：切换武器
 `UpgradeWeapon()`：升级武器
 `Shoot()`：武器射击逻辑

BulletSystem
 功能：子弹系统，处理子弹生成和碰撞检测
 主要方法：
 `CreateBullet()`：创建子弹
 `UpdateBullets()`：更新子弹位置和碰撞检测
 `OnBulletHit()`：处理子弹命中逻辑

ShopSystem
 功能：商店系统，管理商品购买和升级
 主要方法：
 `OpenShop()`：开启商店
 `BuyItem()`：购买商品
 `GenerateShopItems()`：生成商店商品

技术栈

 引擎：Unity 2022.3 LTS
 编程语言：C
 数据驱动：JSON
 物理系统：Unity 2D物理
 UI框架：UGUI
 状态管理：单例模式

游戏玩法

1. 选择一个角色开始游戏
2. 使用WASD键控制角色移动
3. 自动射击，击败敌人
4. 收集敌人掉落的金钱
5. 达到一定波数后，商店会开启
6. 在商店购买武器和升级属性
7. 生存尽可能多的波数

技术亮点

 JSON数据驱动设计：所有游戏数据通过JSON配置，便于内容扩展和修改
 单例模式管理全局状态：GameManager采用单例模式，统一管理游戏状态和资源
 模块化设计：功能模块划分清晰，便于扩展新角色、武器和敌人
 2D物理系统：运用2D物理系统实现流畅的游戏体验
 属性融合机制：通过收集道具提升角色能力
